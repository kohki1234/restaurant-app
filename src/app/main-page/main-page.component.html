Subway Line - Restaurant Search app

Have you ever wondered if you can search some Restaurant according to the metro line?
Then this is the dream tool which you can do what you want!!




Step.1 Search restaurant as you like

You may filter the line name and the app will show you the search result.
Let's say you want to search Restaurant within 南北線, you can select the line from the list, and then you can find all the restaurant!
I prepared some parameter you can manually add to filter the search result. So play around with it!!

Step.2 Utilize map with your search

Also, I integrated Google map with API.
This is going to be really good freind of you. Because you can not only search restaurant with line, but you can also see all restaurant location in Google map!


You can see beautiful pictures of the restaurant. If you want to visit Guru navi site, you can go there from the link!
